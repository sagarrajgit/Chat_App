# Project Name

Chat App

## Description

Chat App created using Flutter/Dart and used firebase for user authentication, firestore for store user's data and fetching realtime data.

# Features
- User Authentication
- Account Creation
- Realtime Chatting

# Libraries
- Firebase
- Firestore
